The Mining Paradise is a dimension containing a dense mountainous environment, mobs and exclusive structures.

Accessing

Main article: Red Ant

In order to enter the Mining Paradise dimenion, the player must find a red ant and right-click on it.


Red ants spawn from red ant nests, so try grabbing some red ant nests and putting them near your base for ease of access.


If cheats are enabled, players can also teleport to the Mining Paradise without right-clicking a red ant: the syntax is /execute in mining_paradise

Environment

The Mining Paradise features mountainous terrain. It contains some stalagmites that contain many ores and also contains unique vegatation.
