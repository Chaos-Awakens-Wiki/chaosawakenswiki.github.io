The Crystal World is a dimension containing many crystal-like blocks, mobs and exclusive structures.

Accessing

Main article: Termite

In order to enter the Crystal World dimenion, the player must find a termite and right-click on it.


If the "crystalWorldRequiresEmptyInventory" config value is set to true (default value is true). Trying to access the Crystal World with items in your inventory will not work.
If the config value has been set to false, you will be able to access the Crystal World dimension with items in your inventory.


Termites spawn from termite nests, so try grabbing some termite nests and putting them near your base for ease of access.


If cheats are enabled, players can also teleport to the Crystal World without right-clicking a termite: the syntax is /execute in crystal_world
