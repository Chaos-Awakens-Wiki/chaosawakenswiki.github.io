Platinum Ore is a extremely rare ore that generates deep underground and is a reliable source of platinum.

Obtaining

Breaking

The platinum ore block can be mined with an amethyst pickaxe, tiger's eye pickaxe or above. When mined, platinum ore will drop itself.

Natural Generation

Platinum ore can also generate in the Mining Paradise in the form of ore features. Platinum ore attempts to generate 3 times per chunk in features of 1-3 ore, between Y levels 1 and 28.


If the "spawnDzOresInOverworld" config value is set to true (default value is false). Platinum ore can generate in the Overworld. Platinum ore attempts to generate 3 times per chunk in features of 1-3 ore, between Y levels 1 to 24.
